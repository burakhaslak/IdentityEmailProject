# üìß BrockMail - .NET Core Identity Email System

BrockMail is a **full-stack web application** designed to mimic a modern email client. It features a solid authentication system, real-time mail statistics, and a sleek, component-based user interface.


---

## üöÄ Features

- Secure Authentication: Powered by ASP.NET Core Identity for user registration, login, and profile management. 
- Email Confirmation System: Users must verify their email addresses using a 6-digit verification code sent via SMTP before they can access the platform.
- Dynamic Dashboard: A data-driven dashboard showing real-time statistics (Inbox, Sent, Drafts, and Unique Contacts) with interactive activity charts.  
- Mail Operations: Star/Unread/Trash status updates, automatic "Save as Draft" functionality, category-based filtering and labeling.
- Modern UI/UX: Built with a component-based architecture using ViewComponents for a modular and maintainable layout.

---

## üõ†Ô∏è Tech Stack

### Backend
- **Framework:** .NET 8.0 / 9.0 (ASP.NET Core MVC)
- **Database:** MS SQL Server with Entity Framework Core
- **Code First Approach**
- **Security:** ASP.NET Core Identity & IConfiguration for secure secret management (SMTP credentials, Connection Strings)
- **Frontend:** HTML5 & CSS3
### Architecture & Practices
- MVC Pattern
- ViewComponent-based UI management
- Clean separation of concerns
- Code First database migrations
